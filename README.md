# ARIBY(Anytime, Anywhere Riding Bicycle)
● 자전거 네비게이션(길찾기, 턴정보, 남은거리, 남은 시간) 기능 제공  
● 사용자 커뮤니케이션(그룹라이딩, 친구, 클럽) 기능 제공  
● 현재 지역의 통합대기정보, 날씨 데이터 제공  
● 서울 지역의 추천 경로 제공 및 길찾기 제공  


# 사용한 라이브러리
● Retrofit2 + OkHttp3  
● Glide  
● Firebase Auth, Database, Storage
